A basic multilevel regression with post-stratification (MRP) model based on some California state polling. Mostly a way for me to play around and get familiar with MRP. 

Polling data from [Berkeley IGS January 2024 poll](https://dlab.berkeley.edu/data/california-public-opinion-polls). Post-stratification census data from [IPUMS](https://usa.ipums.org/usa/). 2020 election data from [MIT Election Lab](https://electionlab.mit.edu/). Some code snippets from the [MRP Case Studies](https://bookdown.org/jl5522/MRP-case-studies/) textbook by Juan Lopez-Martin, Justin H. Phillips, and Andrew Gelman.
